# Веб‑интерфейс

Этот каталог содержит переписанный фронтенд на базе **React**, **Vite** и **Tailwind**.

## Команды
- `npm run dev` — запуск режима разработки
- `npm run build` — сборка статических файлов
- `npm run preview` — просмотр собранной версии
